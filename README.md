# Happylife
Test Rails Engine

Ref. https://guides.rubyonrails.org/engines.html

```bash
rails plugin new happylife --mountable
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'happylife'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install happylife
```
